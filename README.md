# Web API for advertising real estate and services in small/medium-sized towns

## Technologies
- C# .NET
- MySQL Database

## Main Features

### User
- Login
- Logout
- Create profile
- Edit profile
- View user information
- Delete profile

### Properties and Services
- Add
- View details
- Edit
- Delete
- Change status
- Add to favorites
- Search and apply filters
- Contact the advertiser
